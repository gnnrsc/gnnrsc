### 🎓 Master's Thesis

*   **[Digital Twin and Virtual Reality for Ergonomic Analysis](https://webthesis.biblio.polito.it/39926/)**
    Developed a high-fidelity VR Digital Twin of an automotive assembly line using **Unreal Engine**. Integrated **Xsens motion capture suits** to map, track, and record realistic human movements to perform ergonomic analyses and simulations directly within the virtual environment.

### 🚀 Highlighted Portfolio

#### 🥽 VR/AR & 3D Graphics
*   **[DriveAcademyVR](https://github.com/SalvatoreGiugliano98/DriveAcademyVR)** – A Meta Quest 3 driving school application featuring a virtual instructor with real-time vocal feedback. It integrates realistic physics, a diegetic dashboard UI, and haptic feedback to provide a safe, immersive environment for practicing road rules.
*   **[VrBody](https://github.com/gnnrsc/VrBody)** – Virtual Reality project focusing on body tracking and user embodiment mechanics.
*   **[Computer Graphics](https://github.com/SalvatoreGiugliano98/Computer-Graphics)** – Dual project featuring advanced Blender mesh modeling and an interactive 3D game developed from scratch using C++ and OpenGL.

#### 🧠 Machine Learning & Computer Vision
*   **[CanoaSlalom](https://github.com/SalvatoreGiugliano98/CanoaSlalom)** – AI-based analysis of athlete trajectories at the Ivrea Canoa Club slalom stadium. Implemented YOLO (Ultralytics) for real-time tracking and detection to evaluate performance and provide technical support for athletes.
*   **[Face Verification System](https://github.com/SalvatoreGiugliano98/FaceVerification)** – iOS application developed in Unity utilizing iPhone LiDAR sensors for biometric identity verification. It leverages depth-sensing data and deep learning to engineer a robust, spatial-aware facial recognition pipeline.
*   **[MLVM Noise Remover](https://github.com/Frads01/mlvm-noise-remover)** – Engineered an advanced Self-Supervised Learning system for audio denoising using a 20-layer Deep Complex U-Net. By implementing the Noise2Noise methodology, the model learns to reconstruct high-quality musical signals training exclusively on noisy data, overcoming the lack of clean datasets.
